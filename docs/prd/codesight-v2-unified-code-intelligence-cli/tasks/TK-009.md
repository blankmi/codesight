- Source of truth for command input: `cs implements <symbol>` positional argument.
- Invariants:
  - Symbol resolution is name-only via `workspace/symbol`.
  - If multiple definitions match, return deterministic candidate list sorted by file path and fail with:
    - `ambiguous symbol "<name>" — <N> definitions found. Use --path to narrow scope.`
  - No grep fallback is allowed for implements.
  - Missing LSP binary error format is exact:
    - `cs implements: LSP required but <binary> not found. Install: <install command>`
  - Output line contract: `<TypeName> (<path>)`.
  - Final summary line: `<N> implementations`.
  - Ordering is deterministic and de-duplicated.
