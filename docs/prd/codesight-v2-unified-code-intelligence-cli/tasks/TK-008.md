## Data Sources & Invariants
- Source of truth for symbol: first positional argument.
- Source of truth for target project path:
  - `--path` flag when set.
  - otherwise current working directory.
- Source of truth for depth:
  - `--depth` flag when supplied.
  - default `1` when absent.
- Invariants:
  - Command signature remains `cs callers <symbol> [--path <dir>] [--depth <N>]`.
  - Ambiguous symbol message is exact:
    - `ambiguous symbol "<name>" — <N> definitions found. Use --path to narrow scope.`
  - Non-positive depth values fail fast.
  - Command preserves formatter output exactly as produced by `pkg/lsp`.
  - Missing LSP binary error format is exact:
    - `cs callers: LSP required but <binary> not found. Install: <install command>`
